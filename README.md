# AI Symptom Checker 🤖

This project predicts possible diseases based on symptoms using Machine Learning.

## 🔧 Technologies Used
- Python
- Flask
- HTML, CSS, JavaScript
- Decision Tree Algorithm

## 🚀 How to Run
1. Install dependencies:
   pip install flask pandas scikit-learn

2. Run:
   python app.py

3. Open browser:
   http://127.0.0.1:5000

## 📌 Features
- User enters symptoms
- ML model predicts disease
- Simple UI
